  1. SQL INJECTION PAYLOAD
  --------------------------

Payload: 1’ OR ‘1’=’1

Explanation: This payload manipulates the SQL query logic. The condition
‘1’=‘1’ is always TRUE, allowing authentication bypass.

  5. SECURITY LESSON
  --------------------

All attacks were possible because: - No input validation - No output
encoding - Security level set to LOW

Best Practices: - Use prepared statements - Implement Content Security
Policy (CSP) - Sanitize all user inputs - Apply least privilege
principle
